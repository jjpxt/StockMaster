Sobre o projeto

Descrição

Este projeto é uma aplicação web desenvolvida para gerenciar o estoque de um distribuidor de bebidas. O objetivo é facilitar o controle de inventário, permitindo que os funcionários visualizem e atualizem informações sobre os produtos disponíveis.

O sistema foi projetado para ser intuitivo e fácil de usar, com uma interface amigável que permite aos usuários navegar rapidamente entre as seções. A aplicação inclui funcionalidades como visualização de itens em estoque, alertas de quantidade baixa e informações detalhadas sobre cada produto.

O projeto foi desenvolvido utilizando HTML, CSS e JavaScript, com foco na responsividade e usabilidade.
